Información de publicación en Web

Los comentarios del código HTML de esta página contienen la información de configuración que permite a los usuarios modificar páginas del Web con el Asistente para la publicación en Web de Microsoft o con programas que usen el Asistente para publicación en Web de Microsoft, como FrontPad, con el mismo nombre de usuario y la misma contraseña que utilizarían para editar con Microsoft FrontPage. Consulte el SDK de Internet de Microsoft para obtener más información acerca de las API del Asistente para la publicación en Web.
